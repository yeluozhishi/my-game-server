## 基本结构

### 流程
    通过web_center注册或登录账户 -> 获取token、游戏服网关、游戏服列表 -> 选择游戏服 -> 登陆网关 -> 返回获取角色列表 -> 选择角色  
     -> 进入游戏

### web_center  
    承担中控  
    游戏服配置  
    玩家注册登录  
    通过eureka获取游戏服网关列表  
    均衡负载网关

### game_server
    游戏业务


### 协议
    0x0000  选区登录
    0x0001  创建角色
    0x0002  登录角色


### desc
    客户端使用netty连接网关，网关和服务器、服务器与服务器之间kafka通信。
    protobuf通信协议。
    服务器向中控读取配置


### mission
    场景和技能雏形
